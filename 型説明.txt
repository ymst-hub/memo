このテキストについて
    unrealで使われる型について簡単に説明を書きます。
    ctrl + altとかで説明を見れます
    あとググってる

    例内の表例　
        変数名:型名

持ってほしいイメージ
    変数とかを保存、取得するときに、どんな約束で保存したよ～ということを伝えるためのものが型

Boolean型
    true false だけが格納される
    上の2個しか入らないよ！！という約束
    何かを比較したり、チェックしたときに使われる
例）
    中身は一緒かチェック:Boolean = こんにちは:string と こんにちは:string　の比較
        →実行結果
            文字列判定 = true →　結果が正しいよ！！
            文字列判定 = false →　結果が違うよ！！


Byte型
    小さい整数
    マイナスもOK
    Integerでも扱えるが、必要なメモリのサイズが少ないため、軽くなる…はず

Integer型
    整数を扱える
    マイナスもOk
    メインで使う数字の型
    １とかの数字を入れる
    例）
        答え:Integer= 2:Integer + 1:Integer 
        答え = 3


Integer64型
    Integerよりもでかい整数を扱える
    マイナスもOk
    確保するメモリ容量が大きい

Float型
    小数点を扱う
    マイナスもOK

Name型
    変更不可能な文字列
    比較が高速らしい

Text型
    変更不可能な文字列
    翻訳を勝手にしてくれるらしい

String型
    可変な文字列型
    「あいう」とかが当たる

Vector型
    ３つの型（たぶんFloat を 3つ管理している）
    方向とかRGBをまとめて管理できる

Roater型
    回転を扱う型
    TransformのRotation部分のイメージ

Transform型
    位置、回転、スケールをまとめた型（Integer を　9つ管理している）
    Roaterよりも多くの情報を保持している
